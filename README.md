# VIC Tools

## Tools

### bin2ascii
You van find the tool in the folder: 

`bin2ascii`

This tool convert binary vic files into ascii

### VICConvertNetCDF
You van find the tool in the folder: 

`VICConvertNetCDF`

This tool convert from/to seperate ascii/bin files of VIC into NetCDF.

### vic_convert
You van find the tool in the folder: 

`vic_convert`

This tool makes a domain NetCDF file for VIC 5.x
